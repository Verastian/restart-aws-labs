# 🏛️ Prácticas de Arquitectura

Ejercicios de diseño de arquitectura de soluciones sobre AWS: dado un caso (requisitos, restricciones, criterios de éxito), se diseña una arquitectura que lo resuelve y se documenta el razonamiento detrás de cada decisión.

## Metodología

Cada caso combina dos herramientas con propósitos distintos:

- **[D2](https://d2lang.com/)** — para la documentación versionada dentro de este sitio. Los diagramas viven como código (`.d2`) junto al caso, se revisan en el mismo PR que el texto, y quedan embebidos directamente en la página.
- **[Lucidchart](https://lucid.app/)** — para el diseño profundo y colaborativo de la arquitectura (exploración de alternativas, detalle de componentes, anotaciones). Cada caso enlaza directamente al archivo de Lucidchart correspondiente.
